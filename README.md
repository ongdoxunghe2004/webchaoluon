# Hướng Dẫn Sử Dụng - Cháo Lươn Landing Page

Dự án này là một Landing Page tĩnh (Static Website) được xây dựng bằng HTML, CSS, và JavaScript thuần.

## Cách chạy

Vì tất cả mã nguồn nằm trong một file duy nhất, bạn không cần cài đặt bất kỳ công cụ nào (như Node.js hay Python).

1. **Tìm file `index.html`** trong thư mục dự án.
2. **Nhấn đúp (Double click)** vào file `index.html`.
3. Trang web sẽ tự động mở trên trình duyệt mặc định của bạn (Chrome, Edge, Firefox, Safari...).

## Cấu trúc

- `index.html`: Chứa toàn bộ cấu trúc (HTML), giao diện (CSS) và logic (JS) của trang web.

## Lưu ý

- Hình ảnh được sử dụng là các link online (Unsplash) để minh họa. Cần có kết nối internet để hiển thị ảnh.
- Trang web được tối ưu hiển thị tốt trên cả máy tính và điện thoại di động (Responsive).
